## 工具列表

### 1. JSON 美化工具 — `/json/index`
🌐 在线预览：https://ym-wei.github.io/xinNuo/#/json/index

PC 端 JSON 可视化、格式化、双栏对比工具，基于 Vue 3 + `vue-json-pretty`。

**路径**：`dev/src/viewPc/json/index.vue`
**路由**：`/json/index`（页面标题：JSON美化）

#### 功能特性

| 功能 | 说明 |
|---|---|
| ✨ **美化** | 一键格式化 JSON，2 空格缩进，渲染为可交互树形结构 |
| 📦 **压缩** | 去除所有空白，压缩为单行 JSON |
| 🗑️ **清空** | 清空输入与结果 |
| 📋 **复制** | 一键复制美化结果到剪贴板（带 fallback） |
| ↩ **还原输入** | 美化后自动隐藏输入区，可一键还原 |
| 📂 **全部展开/折叠** | 一键切换整棵树的展开折叠状态 |
| 🔀 **双栏对比** | 左右两个独立 JSON 面板，支持拖拽调整宽度 |
| 💾 **自动保存历史** | 美化即自动写入历史（localStorage），去重 upsert，最多 50 条 |

#### 单栏模式（默认）

```
┌────────────────────────────────────────────┐
│ ✏️ 输入 JSON                  120 字符     │
│ ┌────────────────────────────────────────┐ │
│ │  textarea                              │ │
│ └────────────────────────────────────────┘ │
│ ✨ 美化   📦 压缩   🗑️ 清空                │
├────────────────────────────────────────────┤
│ 📄 美化结果  ↩还原  📋复制  📂折叠          │
│ ┌────────────────────────────────────────┐ │
│ │  vue-json-pretty 树形展示              │ │
│ │  (占用剩余高度，可滚动)                  │ │
│ └────────────────────────────────────────┘ │
└────────────────────────────────────────────┘
```

#### 双栏模式

点击顶部 `⇔ 双栏` 切换。每个面板独立：

```
┌──────────────────────┬──────────────────────┐
│ 📄 左侧 · 美化结果    │ 📄 右侧 · 美化结果    │
│ ↩还原 📋复制 📂折叠   │ ↩还原 📋复制 📂折叠   │
│ ┌──────────────────┐ │ ┌──────────────────┐ │
│ │ textarea（输入） │ │ │ textarea（输入） │ │
│ └──────────────────┘ │ └──────────────────┘ │
│ ✨ 美化  🗑️ 清空      │ ✨ 美化  🗑️ 清空      │
│ ┌──────────────────┐ │ ┌──────────────────┐ │
│ │ vue-json-pretty  │ │ │ vue-json-pretty  │
│ │ (占满剩余高度)    │ │ │ (占满剩余高度)    │
│ └──────────────────┘ │ └──────────────────┘ │
└──────────────────────┴──────────────────────┘
       拖拽中间分隔条可调整左右宽度
```

#### 历史记录

- 存储位置：`localStorage['xinNuo_json_history']`
- 上限：50 条
- 点击历史条目 → 自动填入输入框并美化
- 重复内容自动去重（最新操作移到列表顶部）
- 单条删除 / 清空全部

#### 响应式

| 断点 | 行为 |
|---|---|
| `> 1600px` | 内容区加宽至 1400px |
| `≤ 1024px` | 双栏自动改为上下排列 |
| `≤ 768px`  | 紧凑模式，隐藏副标题 |
| `≤ 480px`  | 手机模式，按钮更紧凑 |

#### 键盘快捷键 / 交互

- 树节点支持点击折叠/展开
- 鼠标悬停高亮节点
- 输入框支持垂直拖拽调整高度（限制最大 220px）
- 双栏拖拽分隔条限制范围 20% ~ 80%

#### 技术栈

- Vue 3 (`<script setup>`)
- vue-json-pretty（树形渲染）
- SCSS（样式，scoped）
- localStorage（历史持久化）
- 无第三方 UI 框架，纯自写 UI
